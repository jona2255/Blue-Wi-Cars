Nuestro proyecto

Nuestro proyecto de crédito de síntesis son dos coches teledirigidos, uno con Arduino y el otro sin el Arduino, y los controlamos mediante una aplicación específica para cada coche que nosotros mismo hemos creado.

Para el coche controlado con Arduino, utilizamos Wi-Fi y Bluetooth. La señal enviada mediante Wi-Fi o Bluetooth va a la placa del Arduino, la cual se encarga de procesar la señal y enviar la instrucción al controlador del motor para realizar la acción que le hemos ordenado.

Para el coche que no tiene Arduino, hemos utilizado solo un módulo Wi-Fi con Arduino integrado el cual manda directamente la señal al controlador de los motores.

Las piezas del robot están montadas sobre un chasis que consideramos adecuado, por calidad y precio, para nuestro propósito. Nuestras aplicaciones para controlar los coches solamente están disponible para dispositivos Android.

Nuestro objetivo con este proyecto es el de introducirnos y aprender sobre el manejo de Arduino y también nos da la posibilidad de crear una aplicación para móviles.

  • Montar un robot que se controle por control remoto.

  • Crear una aplicación para android con la que controlar el robot.

  • Hacer que el dispositivo se conecte mediante Wi-Fi y Bluetooth al robot.

  • Comparar el rendimiento, como funcionan y su precio total para saber qué opción es más rentable.

Para este proyecto nos ha hecho falta tanto software como hardware específico para poder montar ambos coches y hacerlos funcionar, al hacer falta instalar librerias para el módulo Wi-Fi en el software de Arduino hemos tenido que utilizar una maquina virtual con una versión más actual.

En cuanto al software utilizado, ha sido más bién poco, hemos usado una maquina virtual con Ubuntu 16.04 Desktop en la que hemos instalado Arduino, una vez teniamos listo Arduino buscamos las librerias que nos hacían falta para el módulo Wi-Fi ESP8266.

Por otro lado, hemos utilizado una variedad de hardware para hacer funcionar ambos robots, cada robot consta de almenos un chasis, dos motores y tres ruedas, a esto hay que sumarle el controlador de los motores y los módulos necesarios.

También nos hemos visto en la necesidad de crear nuestro propio software, en este caso ha sido la aplicación para controlar los robots, desde esta aplicación se pueden controlar ambos ya que está hecha para funcionar tanto por Wi-Fi como por Bluetooth.

El único límite que tiene nuestra aplicación es que necesita estar configurada de igual manera que robot que vamos a controlar, de lo contrario se enviaría un dato que no está declarado en el código del robot y no podria ser interpretado.
